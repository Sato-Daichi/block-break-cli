# block-break-cli

ターミナルで遊べるブロック崩しゲーム

## インストール

```bash
npm install -g block-break-cli
```

## 使い方

```bash
block-break
```

## 操作方法

| キー | 動作 |
|------|------|
| `←` / `a` | パドルを左に移動 |
| `→` / `d` | パドルを右に移動 |
| `Space` | ボール発射 / 一時停止 |
| `q` | 終了 |

## 機能

- 全10レベル（難易度上昇）
- スコア・ライフシステム
- ハイスコア保存
- カラフルなブロック表示

## 開発

```bash
git clone https://github.com/yourname/block-break-cli.git
cd block-break-cli
npm install
npm run build
npm link
```
